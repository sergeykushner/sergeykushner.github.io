# Персональный сайт

## Структура проекта

```
.
├── src/                # Исходный код
│   ├── js/             # JavaScript файлы
│   ├── css/            # CSS файлы
│   └── data/           # Данные (JSON файлы)
├── public/             # Файлы для публикации
│   ├── *.html          # HTML страницы
│   └── sitemap.xml     # Карта сайта
├── assets/             # Статические ресурсы
│   └── badges/         # Бейджи (например, App Store)
└── package.json        # Конфигурация NPM
```

## Разработка

Для локальной разработки открывайте файлы из директории `public/`.

## Сборка

Для сборки сайта (копирование файлов из public в корень для GitHub Pages):

```bash
npm run build
```

## Загрузка активов в Cloudinary

```bash
npm run upload-assets
```
